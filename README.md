# Proposing an ‘unlayered’ neural network model for machine learning"
Individual project by Joseph Parayil, 10th grade, Crooms AOIT

This is a online repository to make certain files, such as Pseudocode, source code, images, notebook, research plan and poster, accessible online.

## Abstract:
This project proposes a new neural network model in response to the current issue that current models are not exactly accurate to the biological nervous systems they draw inspiration from. The proposed model implements spiking functionality for communication between neurons and has no layered topology, taking the form of a random directed graph. This project uses the SDLC, a standard software engineering process, to design and implement this model as a Python class. After rigorously evaluating the coded solution to the validation testing criteria, the solution successfully fulfills the engineering goal and proves that the model is possible in implementation. A future project is envisioned to test the model on a machine learning application to determine the viability of the model for practical use.
